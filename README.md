## Prerequisites
You will need:
- `go` version `1.16` or higher,
- `make`,
- `yarn` with `node.js` version `16` or higher.

## Quick start
Just run `make` from the project root. All dependencies for all modules should
automatically download.

## Project components
- `core` -- backend
- `ui` -- frontend
